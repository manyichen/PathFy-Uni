# PathFy-Uni
# 基于知识图谱的大学生个性化学业规划研究与应用

## 一、项目背景

当前高校人才培养与就业市场需求存在结构性错配，传统学业规划系统面临诸多瓶颈：学生能力评估多采用静态评分机制，无法捕捉能力动态演进趋势；推荐算法依赖简单规则匹配或协同过滤，缺乏对课程、能力、岗位间深层语义关联的挖掘，导致推荐精准度不足且可解释性差；系统交互性薄弱，难以根据学生反馈实时调整规划方案。为响应 "促进高校毕业生更高质量充分就业" 的政策要求，本项目聚焦 "学生 — 岗位 — 校内资源" 协同闭环，打造融合知识图谱与智能推荐技术的个性化学业规划平台，破解人才培养与就业需求 "双向对接" 难题。

## 二、项目目标

构建以 "四维能力画像" 为核心的智能协同平台，通过整合多源数据、构建领域知识图谱、优化智能推荐算法，实现三大核心目标：

- **精准刻画学生能力**：建立学术专业、实践创新、个人素养、社会协作四维能力框架，动态捕捉能力成长趋势与关键跃升节点；
- **量化岗位需求与资源匹配**：构建岗位知识图谱与校内资源知识图谱，实现需求与供给的语义对齐；
- **提供个性化规划服务**：为学生输出短、中、长期学习路径，为导师与管理者提供量化决策支持，助力高校提升人才培养质量与就业指导水平。

## 三、核心技术与创新点

### （一）核心技术栈

- **知识图谱构建**：采用 TransR 算法实现学生 - 课程 - 项目 - 能力异构图、岗位知识图谱、资源知识图谱的嵌入学习，捕捉实体语义关联；
- **动态能力画像**：融合 Hetero-GNN 图神经网络（抽取高阶交互模式）与 Transformer 时间序列模型（捕捉能力演进趋势），解决数据稀疏与新生冷启动问题；
- **智能推荐引擎**：以 IKGCN（交互式知识图谱卷积网络）为核心，结合 LambdaMART 学习排序模型，综合能力差距、提升幅度、学习成本等多维度因素；
- **系统开发**：后端采用 Flask 框架，前端基于 Vue 技术栈，支持多角色访问与可视化交互。

### （二）三大创新点

- **动态能力画像构建**：突破传统静态评估局限，通过知识图谱 + 时序建模，输出含置信区间与贡献度解释的能力成长曲线，实现能力动态追踪与预测；
- **精准可解释推荐**：利用知识图谱语义信息与结构关联，解决冷启动问题，推荐结果可通过图谱路径追溯，让学生理解 "为何推荐"；
- **交互式规划闭环**：支持学生 "采纳 / 修改 / 拒绝" 反馈，实时调整规划方案，结合可视化路线图与预警提醒，实现全学业周期动态陪伴。

## 四、系统核心功能

### （一）学生端

- **四维能力诊断**：生成个性化能力画像，明确与目标岗位的差距及优先提升顺序；
- **定制化规划推荐**：获取短中长期学习路径，包含课程组合、实践安排与预期能力增量；
- **进度跟踪与反馈**：可视化查看成长进度，提交反馈实时优化推荐清单；
- **榜样案例参考**：匹配相似优秀学长案例，借鉴关键成长里程碑。

### （二）导师 / 管理者端

- **学生能力监控**：实时查看学生能力发展动态与规划执行情况；
- **资源优化建议**：基于资源提升效果数据，优化课程设置与实践资源配置；
- **就业指导支持**：获取量化的人才培养效果数据，精准开展就业指导。

## 五、应用价值

- **学生层面**：精准定位能力短板，减少选课与实践的盲目性，提升就业竞争力；
- **高校层面**：实现人才培养数据驱动，优化资源配置，缩小人才培养与市场需求差距；
- **就业层面**：搭建 "学生能力 - 岗位需求" 精准对接桥梁，助力高校毕业生更高质量就业。

## 六、项目预期成果

完成基于 Web 的大学生个性化学业规划系统开发，实现 "数据采集 - 知识建模 - 能力评估 - 推荐规划 - 反馈迭代" 全流程闭环，形成可复用的领域知识图谱与智能推荐算法模型，为高校学业规划与就业指导提供智能化解决方案。
